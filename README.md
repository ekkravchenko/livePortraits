# livePortraits

School of AI (https://courses.learnopencv.com/courses/)

  Hadar Averb\buch-Elor, Daniel Cohen-Or, Johannes Kopf, and Michael F. Cohen. 2017. Bringing Portraits to Life. ACM Trans. Graph. 36, 4
